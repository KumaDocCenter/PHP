# PHP

## 基本信息

**WebSite :**    [PHP](http://php.net/)   [Doc](http://php.net/docs.php)

**维基百科：**   [PHP](https://en.wikipedia.org/wiki/PHP)

**Code：**   [PHP - GitHub](https://github.com/php/php-src)

 

PHP是一种为Web开发而设计的服务器端脚本语言，也被用作通用编程语言。
最初由Rasmus Lerdorf于1994年创建，PHP 参考实现现在由The PHP Group生成。
PHP本来代表Personal Home Page，但它现在代表的缩写 PHP: Hypertext Preprocessor ，中文：PHP(超文本预处理器)。

PHP代码可以嵌入到HTML代码中，也可以与各种网页模板系统，网页内容管理系统和网页框架结合使用。

由Zend引擎驱动的标准PHP解释器是根据PHP许可证发布的免费软件。
自2014年以来，工作已经开始创建一个正式的PHP规范。



**历史：**
PHP开发始于1995年，当拉斯姆斯・勒多夫写了几通用网关接口在C（CGI）程序，这是他用来维持自己的个人主页。

Zeev Suraski和Andi Gutmans在1997年重新编写了解析器，并形成了PHP 3的基础
1998年6月正式发布PHP 3。
Suraski和Gutmans开始重新编写 PHP的核心，于1999年生产Zend Engine。
2004年7月14日，PHP 5发布，由新的Zend Engine II提供支持。
